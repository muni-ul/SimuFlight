# Project brief

AstraLoop is a deterministic Python software-in-the-loop flight-control and validation system for a simplified planar reusable vehicle. Its primary hiring signal is the ability to turn a numerical, stateful, failure-prone system into modular, testable software.

Primary target: systems, simulation, validation, or hardware-adjacent software engineering internships.

Core proof points:

- numerical 2D translation, pitch, and fuel/mass evolution;
- flight software isolated from perfect simulator truth;
- deterministic sensor, controller, actuator, mission, and fault timing;
- configuration-driven nominal and degraded missions;
- tick-aligned telemetry and objective scenario contracts;
- independent numerical-reference tests and layered automated testing.

The project is deliberately local and headless. It does not require network access, hardware, a database, or a hosted UI.
